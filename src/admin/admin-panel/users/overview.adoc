= Overview

**A user** is someone who can log in to the {PRODUCT_NAME} system.

Each user has Email, Full name and Nickname (handle).
Additionally, each user has a username (=email) and password to enable them to log in to the system.
In the list of users an admin can also see the timestamp when user was created and updated.

In the list of users an admin is allowed to:

- Filter users by name or email using search input on the top
- Filter deleted and suspended users
- Edit existing users
- Create new users
