= API clients

*API clients* define an SDK to work with {PRODUCT_NAME} API.
They are provided in the *execution context* (here: `exec(args, ctx)` -- see <<script-execution>>).

.Available {PRODUCT_NAME} API clients:
* https://github.com/cortezaproject/corteza-js/blob/2020.12.x/src/api-clients/system.ts[System]
* https://github.com/cortezaproject/corteza-js/blob/2020.12.x/src/api-clients/messaging.ts[Messaging]
* https://github.com/cortezaproject/corteza-js/blob/2020.12.x/src/api-clients/compose.ts[Compose]
* https://github.com/cortezaproject/corteza-js/blob/2020.12.x/src/api-clients/federation.ts[Federation]
